# Pictitious
### Upload pictures, view pictures uploaded by other users and comment or like it.
### Built using React and Firebase along with Google authentication.

## [Check live here](https://pictitious.netlify.app/)

## Features
- Upload Images
- Like posts
- Comment on posts.
- Mobile responsive
